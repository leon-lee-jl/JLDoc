Chapter 1. Analytics Defined#

With supervised learning you use labeled data, which is a data set that has been classified, to infer a learning algorithm. Unsupervised learning is the opposite of supervised learning, where unlabeled data is used because a training set does not exist.

Fig. 1 Analytics Techniques

Principal components analysis is an Unsupervised Learning technique summarizing a large set of variables and reducing it into a smaller representative variables, called “principal components.”

A fairly common way to investigate or detect intrusions is by using signatures or patterns.
